Task 1: Create and Use an Object

Objective:

Model a real-world entity using an object.

Instructions:

  1. Create an object named book with the following properties:
    • title (string): The name of your favorite book.
    • author (string): The author of the book.
    • pages (number): The number of pages in the book.
    • isRead (boolean): A property indicating whether you have read the book (true or false).
  2. Access and print each property of the book object using both dot notation and bracket notation.
  3. Update the isRead property to true and add a new property called genre with a value of your choice.
  4. Print the updated book object.

Task 2: Create and Modify an Array of Objects

Objective:

Work with an array containing multiple objects.

Instructions:

  1. Create an array named movies containing three objects. Each object should represent a movie with the following properties:
    • title (string)
    • director(string)
    • year(number)
  2. Print the title of the second movie in the array.
  3. Add a new movie object to the end of the movies array.
  4. Update the year of the first movie to 20``23.
  5. Print the entire updated array.

Task 3: Combine Objects and Arrays

Objective:

Use arrays within an object to organize related data.

Instructions:

  1. Create an object named student with the following properties:
    • name (string): Your name.
    • age (number): Your age.
    • subjects (array): An array of three subjects you are studying (e.g., ['Math', 'Science', 'History']).
  2. Print the first subject in the subjects array.
  3. Add a new subject to the subjects array.
  4. Print the updated student object.

Task 4: Challenge Task (Optional)

Objective:

Practice applying objects and arrays to solve a problem.

Instructions:

  1. Create an object named recipe with the following properties:
    • name (string): The name of a dish (e.g., “Pasta Salad”).
    • ingredients (array): An array of ingredient objects. Each object should have:
      • name (string): The name of the ingredient.
      • quantity (string): The quantity required (e.g., “1 cup”).
    • isVegetarian (boolean): Whether the recipe is vegetarian (true or false).
  2. Add a new ingredient to the ingredients array.
  3. Print the name of the second ingredient in the ingredients array.
  4. Print the entire recipe object.
